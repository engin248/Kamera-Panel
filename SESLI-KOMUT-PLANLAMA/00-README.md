## ⚙️ PLAN YAZMA KURALLARI (Herkese)

```
1. Önce MİSYON'u oku (bu dosyanın başında)
2. BRIEFING'i kelime kelime oku
3. Diğer planları oku — fikirlerini çal değil, geliştir
4. Kendi planını TAMAMEN bağımsız yaz
5. Planın şunları içermeli:
   - Teknoloji seçimi ve nedeni
   - Mimari (nasıl kurulacak)
   - İş bölümü (kim ne yapacak)
   - Aşamalar (önce ne, sonra ne)
   - Riskler
   - Tavsiye (en iyi yol)
6. Planın sonuna GK kodunu yaz
7. Kod yazma — sadece plan
8. Sisteme dokunma — sadece plan
```
