# KISIM A: TABLET SEÇİMİ

## 1. İHTİYAÇ ANALİZİ

Tablet ile yapılacak işlemler:
- ✅ İlk ürün hazırlanırken **video çekmek** (her işlem ayrı ayrı)
- ✅ **Sesli anlatım kaydetmek** (her işlemin yapılışı)
- ✅ **Fotoğraf çekmek** (işlem karşılaştırma)
- ✅ Seri üretimde **video izlemek** (talimat videoları)
- ✅ **Ses dinlemek** (sesli talimatlar)
- ✅ **Yazılı talimat okumak**
- ✅ **Manuel veri girmek** (başlama/bitiş saati, adet, OK/Red)
- ✅ Toplanan verileri **depolamak** ve sonra bilgisayara aktarmak
- ✅ Fabrika/atölye ortamında **dayanıklı** kullanım

# KISIM B: PİLOT UYGULAMA PLANI (Pazartesi Başlangıç)

## 1. Pilot Uygulama Yaklaşımı

```
PAZARTESİ BAŞLANGIÇ
        │
        ▼
┌─────────────────────────────────────────────┐
│  AŞAMA 0: MANUEL VERİ TOPLAMA              │
│  (Sistem hazır olana kadar)                 │
│                                             │
│  Araç: Tablet + Kamera uygulaması           │
│  Kayıt: Video/ses/fotoğraf → tablet hafıza  │
│  Veri girişi: Manuel (Excel veya form)      │
│  Aktarım: USB ile bilgisayara günlük yedek  │
└─────────────────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────┐
│  AŞAMA 1: BASİT WEB UYGULAMASI (1-2 hafta) │
│                                             │
│  → Model/İşlem tanımlama                    │
│  → Video/ses yükleme                        │
│  → Operatör talimat ekranı                  │
│  → Manuel başla/bitir/OK kayıtları          │
│  → Basit raporlar                          │
└─────────────────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────┐
│  AŞAMA 2: OTOMATİK SİSTEM (3-4 hafta)      │
│                                             │
│  → Otomatik zaman hesaplama                 │
│  → Performans raporları                     │
│  → Ücret/prim hesaplama                     │
│  → Kamera entegrasyonu                      │
└─────────────────────────────────────────────┘
```

## 2. Pazartesi Günü Manuel Yapılacaklar

### Prototip/İlk Ürün Aşaması:
1. Tableti aç, **kamera uygulamasını** başlat
2. Model makinacı ilk ürünü hazırlarken:
   - Her işlemi **ayrı video** olarak kaydet
   - Her videoda **sesli anlat**: Ne yapıldı, nasıl yapıldı, nelere dikkat edilmeli
   - İşlem bittikten sonra **fotoğraf** çek
3. Her işlem için not al:
   - İşlem adı
   - İşlem sırası
   - Zorluk derecesi (1-5 veya 1-10)
   - Tahmini süre
4. Notları **Excel'e** gir (veya kağıda yazıp sonra gir)

### Seri Üretim Aşaması:
1. Her operatöre işlemini tabletten **video olarak göster**
2. Operatör ilk ürünü yaptığında **fotoğraf çek**, karşılaştır
3. **Başlama saatini** not et
4. Her 20 üründe bir **ara kontrol** — fotoğraf çek, OK/Red not et
5. **Bitiş saatini** ve **toplam adeti** not et
6. Akşam tüm verileri **Excel'e gir**
7. Video ve fotoğrafları **bilgisayara USB ile aktar**

# KISIM C: SİSTEM GELİŞTİRME YOL HARİTASI

## 1. Teknoloji Kararı

### Seçilen Mimari: Next.js (React) + SQLite → PostgreSQL

| Karar | Seçim | Neden |
|-------|-------|-------|
| **Dil** | JavaScript/TypeScript | En yaygın web dili, en çok kaynak var |
| **Framework** | Next.js 14+ (React) | Hem sunucu hem istemci, hızlı, modern |
| **Veritabanı (Başlangıç)** | SQLite | Kurulum gerektirmez, basit, hızlı başlangıç |
| **Veritabanı (İleri)** | PostgreSQL | Çok kullanıcılı, güçlü, ölçeklenebilir |
| **Dosya Depolama** | Yerel klasör | Basit, hızlı, sonra buluta taşınır |
| **UI Bileşenleri** | shadcn/ui + Tailwind CSS | Profesyonel görünüm, hızlı geliştirme |
| **Dil Desteği** | next-intl | Türkçe + Arapça + genişletilebilir |
| **Kimlik Doğrulama** | NextAuth.js | Basit, güvenli kullanıcı girişi |
| **Raporlama** | ExcelJS | Excel dosyası oluşturma/dışa aktarım |
