# 01 — PROBLEM TANIMI

## Tekstil Atölyesinde Yaşanan Gerçek Sorunlar

**Kaynak:** Komutan Engin Bey (sahadan)
**Tarih:** 01 Mart 2026

---

## ANA PROBLEM

Tekstil atölyesi zarar ediyor.
İşletme sahibi orada olmadığında sistem çalışmıyor.
Kimin ne yaptığı belli değil → Adil ücret ödenemiyor.

---

## 4 TEMEL BÖLÜMÜN SORUNLARI

### BÖLÜM 1 — MODEL YÖNETİMİ

```
Sorun: Firmadan gelen teknik dosya kağıtta kalıyor
       Ölçüler, fotoğraflar, işlemler sisteme girmiyor
       Model standartları kayıt altında değil
       Hangi ürün kaç parçadan oluşuyor → belli değil
```

### BÖLÜM 2 — ÜRETİM SÜRECİ

```
Sorun: Parti geldiğinde eksikler kontrol edilmiyor
       Kim açtı, ne zaman açtı → kayıt yok
       İlk ürün nasıl dikildi → kayıt yok
       İşlem sırası belli değil → yanlış personele iş gidiyor
       Seri üretimde başlangıç/bitiş saati → belli değil
       Temizleme/paket aşamaları → kayıt yok
```

### BÖLÜM 3 — MALİYET

```
Sorun: Saatlik işletme maliyeti hesaplanmıyor
       Ürün başına maliyet belli değil
       Fason sipariş kâr mı zarar mı belli değil
       (Dikilip bittikten sonra anlaşılıyor — çok geç)
```

### BÖLÜM 4 — PERSONEL & ÜCRET

```
Sorun: Giriş/çıkış saatleri takip edilmiyor
       Mesai elle hesaplanıyor → hatalı
       Kim kaç adet üretmiş → belli değil
       Prim sistemi yok → adaletsiz ücret
       Çalışmayan çalışana yük bindiriyor
```
